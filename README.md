# BTC Curate Assistant

CLI tooling for BTC metadata curation.

```bash
uv run curate
```

## Docs

- [BTC Curate Assistant](tools/btc-curate-assistant.md) — full setup and usage guide for the curation CLI
- [Coding Agents](tools/coding-agents.md) — installing Claude Code and OpenAI Codex on Linux
- [dash_curation venv (UV)](tools/dash-curation-venv-UV.md) — setting up the dash_curation app environment with UV

## Repo Layout

- `src/btc_manifest/` — curation CLI source
- `templates/` — source `.xlsx` manifest templates
- `tools/` — setup and reference docs
- `files/` — gitignored project/reference files
- `runs/` — gitignored generated curation runs
